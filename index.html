HARDWARE SECURITY

Hardware-Enabled Root of Trust

A Root of Trust (RoT) anchors security in immutable hardware—physically isolated from the CPU, with its own cryptographic engines and factory-burned keys. Unlike software roots of trust, it cannot be subverted even by an attacker with full OS access.

Hardware-Backed Identity

WebAuthn API leverages your device TPM, Windows Hello, or YubiKey. Private keys never leave the hardware boundary. Cryptographic signatures are verified in real-time using WebCrypto APIs.

Boot Integrity & Attestation

Measured Boot extends PCRs cryptographically at each boot stage: PCRnew = SHA256(PCRold ‖ Hash). Remote Attestation compares these against golden baselines in Firebase to detect tampering.

Sealed Storage

Encryption keys are sealed to specific PCR states. If the boot chain is tampered with, the TPM hardware refuses to unseal the key — even physical access cannot extract it.

Hardware Authentication

Register your hardware security key, Windows Hello, or platform biometric. Private keys are generated inside the secure hardware and never exported.
